The Call Taxi Booking Application is a console-based Java program that simulates a real-time taxi booking system. It manages four taxis operating along fixed points A to F, with each point separated by 15 kilometers. All taxis are initially stationed at point A and are allocated to customers based on availability, nearest location, and lowest total earnings. The fare is calculated as Rs.100 for the first 5 kilometers and Rs.10 for each additional kilometer traveled. The system ensures that taxis are charged only from the pickup point to the drop point. If no taxi is available at the requested pickup time, the booking is automatically rejected. The application also provides a feature to display complete taxi details, including total earnings and booking history.
